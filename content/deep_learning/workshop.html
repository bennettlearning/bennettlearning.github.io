Pre-requisite and Installation Instructions

Anaconda 5.1 - Python Version 3.6 You can Download it from the link freely from the given link.

Click here

Python Libraries: numpy, pandas, sklearn, nltk, tensorflow, keras and for plotting matplotlib


Installation of Packages:

For installing the above mentioned python packages please follow the procedure mentioned below.

1. For installing new packages, system must be connected through internet.

2. Press start in windows and search for "Anaconda Prompt". When you will click on the application, a prompt window will be open,wait for few seconds.

3.You can check/ install any package in Anaconda Python by..
      pip install package_name

f-ex:
pip install numpy
and press Enter key after typing these commands.

Note. If you are installing numpy like this, it will give requirement already satisfied message on the prompt. that means package is already installed in the anaconda python.

3. Installing Opencv for Image processing

Start the Anaconda Prompt: Press start in windows and search for "Anaconda Prompt". when you will click on the application, a prompt window will be open, wait for few seconds.

type command:
pip install opencv-python
press Enter.

4. Installing Tensorflow and Keras:

Start the Anaconda Prompt in administrator mode: Press start in “windows”, type “Anaconda Prompt”, and right click on the application and click “Run as Administrator”.

Repeat step 2
f.ex- pip install tensorflow
pip install keras

5. Installing nltk and corpus: For Natural Language Processing:

a. Start the Anaconda Prompt in administrator mode: Press start in “windows”, type “Anaconda Prompt”, and right click on the application and click “Run as Administrator”.

Type on Anaconda Prompt as:
pip install nltk()
Press Enter.

b. Go to the start menu, search for Jupyter notebook. Click on it. Wait for a moment and a web browser will be open with home screen. Right part of this browser click on “New” keyword select for python 3.

It will open a python 3 notebook. Then follow the given below instruction.

1 Type command

Import nltk

nltk.download(‘popular’)

2. Run the file, it will automatically download all the important packages inside the nltk toolkit.

Help: You can also take help of YouTube video: for sample installation of Anaconda + Tensorflow + Keras

Note: If you want to work with regular python installed from (https://www.python.org/downloads/), installation of numpy, pandas, sklearn, ntlk, tensorflow, keras libraries can be still done using “pip install” command similar to step 2 and 3. However, installation of Anaconda Python is highly encouraged for the lab session.
